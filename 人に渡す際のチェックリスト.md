# 人に渡す際のチェックリスト

このプログラムを上司や他の人に渡す前に、以下のチェックリストを確認してください。

## ✅ 必須チェック項目

### 1. 機密情報の確認

- [ ] **`service_account.json` が含まれていないことを確認**
  - このファイルは `.gitignore` で除外されています
  - Gitリポジトリに含まれていないことを確認：`git ls-files | grep service_account`
  - ZIPファイルで渡す場合も、このファイルは**含めない**でください

- [ ] **APIキーやパスワードがコードにハードコードされていないことを確認**
  - `main.py` 内に機密情報が含まれていないか確認済み ✅

### 2. ドキュメントの確認

- [ ] **`README.md`** が最新であること
- [ ] **`初回セットアップガイド.md`** が含まれていること
- [ ] **`自動化セットアップ手順.md`** が含まれていること（自動実行を使用する場合）
- [ ] **`requirements.txt`** が正しく設定されていること

### 3. 設定ファイルの確認

- [ ] **`com.cme.scraper.plist`** のテンプレート（`.template` ファイル）を作成済み
- [ ] 実際の `com.cme.scraper.plist` に絶対パスが含まれている場合、テンプレート版を推奨

### 4. 動作確認

- [ ] **手動実行が正常に動作することを確認**
  ```bash
  source venv/bin/activate
  python main.py
  ```

- [ ] **スプレッドシートにデータが正しく書き込まれることを確認**

- [ ] **ログファイルが正常に生成されることを確認**

## 📦 渡し方の選択肢

### オプションA: Gitリポジトリからクローン（推奨）

```bash
git clone https://github.com/yousunafu/CME.git
cd CME
```

**メリット：**
- 最新版を常に取得できる
- 更新が簡単
- バージョン管理が明確

**必要な情報：**
- リポジトリURL: `https://github.com/yousunafu/CME.git`
- ブランチ名（通常は `main`）

### オプションB: ZIPファイルで渡す

1. 以下のファイルとディレクトリを含める：
   ```
   CME/
   ├── main.py
   ├── requirements.txt
   ├── README.md
   ├── 初回セットアップガイド.md
   ├── 自動化セットアップ手順.md
   ├── 人に渡す際のチェックリスト.md
   ├── com.cme.scraper.plist.template
   ├── .gitignore
   └── logs/（空のディレクトリ、または .gitkeep ファイル）
   ```

2. **以下のファイルは含めない：**
   - `service_account.json` （機密情報）
   - `venv/` （仮想環境は各自で作成）
   - `__pycache__/` （Pythonのキャッシュ）
   - `.DS_Store` （macOSの設定ファイル）

**ZIPファイル作成コマンド例：**

```bash
# 必要なファイルのみを含むZIPを作成
zip -r CME-for-transfer.zip \
  main.py \
  requirements.txt \
  README.md \
  初回セットアップガイド.md \
  自動化セットアップ手順.md \
  人に渡す際のチェックリスト.md \
  com.cme.scraper.plist.template \
  .gitignore \
  -x "*.pyc" "__pycache__/*" "*.DS_Store"
```

## 📝 渡す際に伝えるべき情報

### 必須情報

1. **初回セットアップが必要であること**
   - `初回セットアップガイド.md` を最初に読むように伝える

2. **Google Cloud Project の設定が必要であること**
   - サービスアカウントの作成方法
   - 認証キーの取得方法
   - スプレッドシートの共有設定方法

3. **スプレッドシート名**
   - デフォルト: "CME定期調査"
   - 変更する場合は `main.py` の28行目の `SPREADSHEET_NAME` 定数を編集する必要がある

### 推奨情報

4. **環境要件**
   - Python 3.8以上
   - macOS（自動実行を使用する場合）
   - インターネット接続

5. **トラブルシューティング**
   - `初回セットアップガイド.md` のトラブルシューティングセクションを参照

## 🔒 セキュリティに関する注意事項

渡す際に、以下の点を必ず伝えてください：

1. **`service_account.json` は機密情報**
   - 他人と共有しない
   - GitHubなどの公開リポジトリにコミットしない
   - 誤ってコミットした場合は、すぐにGoogle Cloud Consoleでキーを無効化する

2. **個人アクセストークン（もし使用している場合）**
   - GitHubのPersonal Access Tokenを使用している場合、同様に機密情報として扱う
